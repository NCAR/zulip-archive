@Michaela Sizemore and our SciPARCS interns @Rose Craker and @Claire Fiorino have been developing a growing gallery of Python plots by systematically going through the most used NCL examples. If anyone is interested you can see the gallery here: https://geocat-examples.readthedocs.io/en/latest/gallery/index.html

We have been discussing interesting questions such as when to copy the NCL example 1-to-1 in appearance and when to use more current standards (no more rainbow color maps!), how to make these plots more searchable by feature, and most importantly when to write new functions for geocat-viz or geocat-comp based on common workflows or unique NCL functions we encounter throughout the process.

@John Clyne @Kevin Paul @Orhan Eroglu and I are all really proud of our students' work towards the progress and growth of this gallery, and we feel it hosts enough diverse examples that it is a potentially a valuable resource for anybody working in Python.

Please let us know if you have opinions on what kinds of plots we should prioritize as the gallery continues to expand.
