The Symposium of Advances in Modeling and Analysis Using Python is now accepting abstracts. This symposium takes place during the annual AMS meeting which will be in New Orleans January 10-14.

We're looking for abstracts in these topics:
Working with large data sets using Python
Visualization and data discovery using Python
Teaching, training, outreach and building communities around Python
Python in operations and research to operations
New Python tools in the atmospheric and oceanographic sciences

In addition there will be joint sessions on:
Workflows for cloud-based data analysis in Python with a focus on CMIP6 (Python and the 34th Conference on Climate Variability & Change)
Python frameworks for reproducible communicable Artificial Intelligence workflows (Python and Conference on AI)
Exploring the Scalability and Parallel Performance of Python on High Performance Computing Systems (HPC and Python)

Abstract are due August 3rd and you can find the link to submit your abstract here:
https://annual.ametsoc.org/index.cfm/2021/program-events/conferences-and-symposia/11th-symposium-on-advances-in-modeling-and-analysis-using-python/
