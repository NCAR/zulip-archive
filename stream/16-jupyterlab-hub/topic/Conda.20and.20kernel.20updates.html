Topic: Conda and kernel updates

Brian Vanderwende (Jan 13 2024 at 17:45):

Heads up on some conda/kernel updates: we've done our bi-yearly update of conda and mamba that are provided via the conda module. conda now uses libmamba as the solver, so hopefully environment creation and package installation will be faster if not using the mamba command.

We've also deployed the latest NCAR Python Library (NPL 2024a), available via the module and in JupyterHub as a kernel. This version bumps Python to 3.11. Note that two packages do not currently have 3.11 support via conda-forge, and so they have been dropped from this version of the NPL: pyspharm and windspharm .

More new language kernel versions will come in the next week or two.
