Topic: WACCM diagnostics in ADF?

Michael Levy (Jan 18 2024 at 20:48):

@Teagan King was asked about whether CUPiD would support CESM cases with WACCM / WACCM-X. Is ADF CAM-specific, or will it work with WACCM?

Brian Medeiros (Jan 18 2024 at 21:00):

ADF will work with WACCM. There are a few things that might need some user attention if really focusing on WACCM results. The main one is to switch the latitude-pressure (or longitude-pressure) plots to use a log-pressure scale. I think we currently control this in the variable defaults yaml file with a flag. The other thing would be to adapt contour levels to be appropriate if plotting quantities at pressure levels above the troposphere. That's also controlled on a per-variable basis in the variable defaults file.

Jesse Nusbaumer (Jan 18 2024 at 23:20):

I just wanted to add that @Justin Richling has also added a few diagnostics that are designed for higher-top versions of the model like WACCM. Specifically they are diagnostics for the QBO, the Transformed Eulerian Mean (TEM) circulation, and the "tape recorder" plots for troposphere->stratosphere water vapor transport. I am sure Justin can tell you more if you'd like!

Jesse Nusbaumer (Jan 18 2024 at 23:21):

And just to be clear those are all diagnostics that can now run automatically in the ADF.
