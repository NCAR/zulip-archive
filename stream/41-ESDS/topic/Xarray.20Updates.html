Deepak Cherian (Jul 22 2022 at 17:20):

We've just released v2022.06.0!

This release brings

  • a major internal refactor of the indexing functionality. This will allow easier indexing of curvilinear grids, unstructured grids, and accounting for bounds variable in the future
  • the use of flox for faster groupby operations (though dask improvements need some opt-in keyword arguments)
  • and experimental support for the new Python Array API standard.

See the release notes for a full list of changes.
