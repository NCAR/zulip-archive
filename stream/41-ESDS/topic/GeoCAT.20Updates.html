Orhan Eroglu (Feb 19 2021 at 18:39):

GeoCAT team has released v2021.02.0 of GeoCAT-comp (computational component) on conda package manager.

This release has a number of new functions implemented (such as climatology, anomaly, relhum, dewtemp, etc.) as well as existing functions refactored to modern Python implementation and documentation (such as EOF analysis).

Please check the user API and so for a full list of functionality:

GeoCAT-comp User API

Orhan Eroglu (Feb 19 2021 at 18:46):

GeoCAT team is planning to employ a continuous delivery model with quasi-monthly releases as well as one or two major releases in 2021.

The next release will be around late-February or early-March, and the team is planning to have the following important function implemented in Python (In addition to code and docs refactoring as well as potential other new functions)

vinthp - Vertical Interpolation of Hybrid to Pressure Coordinates

Please feel free to give your feedbacks and offer contribution!
